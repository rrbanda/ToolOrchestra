# Maintainers

orchestrator-rhoai is a production-grade open-source reference implementation that deploys NVIDIA's ToolOrchestra system on Red Hat OpenShift AI. This document lists maintainers by area of ownership and describes how to become one.

| Area | Description | Maintainer(s) |
|------|-------------|---------------|
| **Kubernetes manifests** | OpenShift/Kubernetes YAML, Kustomize overlays, Helm charts, RBAC, NetworkPolicies | TBD |
| **Orchestrator gateway** | API gateway, routing, auth, request handling | TBD |
| **Evaluation scripts** | Benchmarking, evaluation pipelines, metrics | TBD |
| **Training pipeline** | Model training, fine-tuning, dataset handling | TBD |
| **Documentation** | README, guides, API docs, architecture docs | TBD |
| **CI/CD** | GitHub Actions, image builds, release automation | TBD |

Maintainers may own multiple areas. As the project grows, additional areas may be defined.

## How to Become a Maintainer

1. **Contribute consistently**: Submit high-quality PRs, reviews, and issue triage over several months.
2. **Demonstrate domain expertise**: Show familiarity with your area(s) of interest (manifests, gateway, evaluation, training, docs, CI/CD).
3. **Exhibit good judgment**: Make decisions that align with project goals, security, and maintainability.
4. **Be reliable**: Respond to issues and PRs in a timely manner; follow through on commitments.
5. **Get nominated**: An existing maintainer nominates you based on the above.
6. **Gain consensus**: Other maintainers approve the nomination.
7. **Accept responsibility**: Agree to uphold the [Code of Conduct](CODE_OF_CONDUCT.md), [Governance](GOVERNANCE.md), and [Security](SECURITY.md) practices.

After joining, new maintainers receive:
- Write/merge access to the repository
- Invitation to maintainer discussions
- Assignment to one or more areas of ownership in this document

## Stepping Down

Maintainers may step down at any time by notifying the team. They will be removed from the roster but remain welcome as contributors. Access and ownership will be reassigned as needed.
